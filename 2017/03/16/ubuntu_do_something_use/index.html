日常使用软件的安装与配置

更新更新源为国内源

安装好系统的第一件事，就应该换源！！！！除非你在一个不被墙的地方
在系统设置里的软件和更新中选择合适的国内源

虽然ubuntu自带很多万能驱动，但是对于显卡之类的来说，还是专门的驱动比较好用
所以在系统设置里的软件和更新中附加驱动选择专有驱动并保存

然后重启就好
