运维
  • Docker

在选择本地git服务时，通常大家都会选择直接安装gitlab，但是其安装复杂，占用资源多等缺点有点让人接受不了，肯定不想一台服务器只用来跑gitlab了，有些太浪费。
还可以选择直接在系统上安装没有图形界面的git服务器，但是带来了不易查看等缺点
